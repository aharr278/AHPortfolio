Introduction

Good morning, afternoon, or evening! Welcome to my portfolio/final project for CVEN 5390. I have collected multiple maps and styles created in MapBox for other assignments or for this final project on this site. The focus on this site is creating maps around climate change and environmental concerns while also addressing user disabilities for these maps like color blindness. Please enjoy!

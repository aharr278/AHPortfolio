Description

The purpose of this section is to create and display maps which are not specifically intended for a single set of users but are viewable and understandable for users both with and without various visual kinds color blindess. For more information on how the color palette was determined please reference: Palette Base. The Okabe_Ito palette created by Joachim Goedhart was used for these maps.

User Stories

  • View locations across the world with the color palette
  • Select a spot on the map for color information
  • Zoom in and out of the map
  • View the information in a color blind friendly format
  • Geographical information distinguishable for those with different color deficiencies
  • The user can find the data source for the map
  • The user can print out the map as an example

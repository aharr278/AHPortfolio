Description

Impaired water simply means that the quality of water is not clean enough for its designated use. Use can be drinking, watering, cleaning, and more. The EPA lists different datasets containing information about these impairments, what bodies of water they are and efforts in place to restore them: EPA. The purpose of these maps is to present the bodies of water in the U.S. which are impaired in color-blind friendly formats.

User Stories

  • Select a water point location for more information

  • Select a water point on the map to see a link for more information
  • Select a water point to see when it was recorded
  • Search for specific water points on the map
  • The user can find the data source for the map
  • The user can print out the map as an example
  • Compare water locations to states
  • Compare water locations to cities
  • View the information in color blind friendly formats
  • Geographical information distinguishable for those with different color deficiencies
